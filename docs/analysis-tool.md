# Statistical Analysis & Visualization Tool (`tool.py`)

`tool.py` is a desktop GUI application for post-processing the CSV outputs generated by the SickleSight analysis pipelines. It produces publication-quality violin plots comparing morphological metrics (Aspect Ratio, Eccentricity, Circularity) between sickled and non-sickled cell populations, with automated statistical annotations.

---

## Requirements

`tool.py` is included in the standard `sicklesight` Conda environment. Activate it before running:

```bash
conda activate sicklesight
python tool.py
```

No additional dependencies are needed beyond those already specified in `sicklesight_env.yaml`.

---

## Overview of the Interface

The application window is divided into three main areas:

```
┌──────────────────────────────────────────────────────────────────┐
│  Top bar: Metric selector  |  Plot options  |  Y-axis range       │
├────────────────────────────────────┬─────────────────────────────┤
│  Group panels (scrollable)         │  Plot canvas                │
│  ┌─────────┐ ┌─────────┐           │                             │
│  │ Group A │ │ Group B │  ...       │  [ Violin / box plot ]      │
│  └─────────┘ └─────────┘           │                             │

│  Bottom bar: Plot individual | Plot composite | Save/Load config  │
```

## Input Data Format

`tool.py` reads the per-frame CSV files produced by `sicklesight_part2.py` or `sicklesight_merged.py`.

**Required columns:**

| Column | Type | Description |
|--------|------|-------------|
| `Sickle_Label` | int | Cell state: `0` = sickled, `1` = non-sickled |
| `Frame_Index` | int | Frame number within the video |
| `Class_ID` | int | Morphological class (0–6 → A–G) |

**Optional columns (needed for the corresponding metric):**

| Column | Type | Description |
|--------|------|-------------|
| `Aspect_Ratio` | float | Major axis / minor axis |
| `Eccentricity` | float | 0 (circle) → 1 (line) |
| `Circularity` | float | 4π × Area / Perimeter² |

These files are typically named `frame<N>_raw_data.csv` inside each video's output folder.

## Step-by-Step Usage

### 1. Select a metric

In the top bar, choose one of:
- **Aspect Ratio (AR)**
- **Eccentricity (ECC)**
- **Circularity**

All plots and statistics will use the selected metric.

### 2. Load CSV files into groups

The tool supports up to **4 groups** (e.g., four experimental conditions or concentrations). For each group:

1. Click **Load CSV** inside that group's panel.
2. Navigate to and select the `frame<N>_raw_data.csv` file for that condition.
3. The group status indicator turns green once a file is loaded successfully.

### 3. Select a frame

Each group panel shows a **Frame** dropdown. After loading a CSV, the dropdown is populated with all available frame indices found in the file.

- Frames that are currently selected for plotting are marked with a **✓**.
- Choose **Normal** or **Sickle** from the toggle to filter which cell-state population is shown.

### 4. Filter by morphological class

Use the **Class** multi-select dropdown to include or exclude specific morphological classes (A–G) from the plot. By default all classes are selected.

- Click **Select All** or **Deselect All** for quick toggling.

### 5. Configure plot options

In the top bar:

| Option | Description |
|--------|-------------|
| Show legend | Display a sample-count legend on the plot |
| Show threshold | Draw a horizontal reference line (AR: 1.6; others: configurable) |
| Show stats | Overlay Mann–Whitney U test results |
| Simplify p-value | Show significance stars (*, **, ***) instead of exact p-values |
| Stats location | **Legend** — embed stats in the legend; **Bracket** — draw bracket annotations on the plot |
| Y-axis min / max | Set custom Y-axis range (leave blank for auto-scale) |
| Sickle label / Non-sickle label | Customize axis tick labels |

### 6. Generate plots

**Plot individual group:**
- Click the **Plot** button inside a group panel to generate a single violin plot for that group.

**Plot composite figure:**
- Click **Plot Composite** in the bottom bar to combine all loaded groups into a single multi-panel figure, suitable for direct use in publications.

Plots are displayed in the canvas on the right side of the window. Use the toolbar beneath the canvas to zoom, pan, or save the figure as a PNG or PDF.

---

## Statistical Annotations

When **Show stats** is enabled, the tool automatically runs **Mann–Whitney U tests** comparing the sickled vs. non-sickled distributions for each group/frame combination and annotates the plot:

| p-value | Displayed as |
|---------|--------------|
| > 0.05 | ns |
| ≤ 0.05 | * |
| ≤ 0.01 | ** |
| ≤ 0.001 | *** |
| ≤ 0.0001 | **** |

In **Bracket** mode, horizontal brackets are drawn between compared pairs with the significance notation above each bracket. The bracket positions are computed automatically to avoid overlaps (skyline algorithm).

---

## Saving and Loading Configuration

Once you have set up groups, frames, class filters, and plot options, save the configuration so you can reload it later:

- Click **Save Config** → choose a `.json` file path.
- Click **Load Config** → select a previously saved `.json` file to restore the full session state.

This is especially useful for reproducing figures or sharing analysis settings with collaborators.

---

## Undo / Redo

Use the **Undo** and **Redo** buttons in the bottom bar (or `Ctrl+Z` / `Ctrl+Y`) to step backward or forward through changes made to the group data and settings. A history log is shown in the History pane.

## Typical Workflow

```
1. Run sicklesight_part2.py or sicklesight_merged.py to generate CSVs
2. conda activate sicklesight && python tool.py
3. Select metric: e.g. "Aspect Ratio"
4. Load frame0_raw_data.csv into Group A (condition 1)
5. Load frame0_raw_data.csv into Group B (condition 2)  [different experiment]
6. Select target frame from each group's Frame dropdown
7. Enable "Show stats" + "Bracket" mode
8. Click "Plot Composite"
9. Save figure via the canvas toolbar
10. Optionally: Save Config → share .json with collaborators
```

---

## Output

All plots are rendered interactively on the canvas. To save:

- Use the **floppy disk / save icon** in the Matplotlib toolbar below the canvas.
- Supported formats: PNG, PDF, SVG, EPS (depending on your Matplotlib backend).

There is no automatic file output — plots must be saved manually from the canvas toolbar.

## Troubleshooting

| Issue | Solution |
|-------|---------|
| CSV loads but frame dropdown is empty | Check that the CSV contains a `Frame_Index` column with valid integer values |
| Plot is blank after clicking Plot | Verify that the selected frame exists in the loaded CSV and that at least one class is selected |
| Stats annotations overlap | Reduce the number of comparison groups, or switch from Bracket to Legend mode |
| GUI is unresponsive | Large CSVs with many frames can take a few seconds to parse; wait for the status indicator to turn green |
| `ModuleNotFoundError` | Ensure you are running inside the `sicklesight` Conda environment |
